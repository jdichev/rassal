Using Rassal for presentations

A minimalistic tool for making simple presentations

Inspired by the Takahashi method and Big

Based on large letters and impactful typography

Based on large letters and impactful typography

Why a separate tool

Easy. For fun and practice

It uses simple HTML elements, like:

  • Headings
  • Unordered lists
  • Ordered lists
  • Paragraphs

What it does is navigation and styling

By default slides are expanded to maximum font size for available space

Content is responsive to viewport

Navigation between slides with arrow keys
